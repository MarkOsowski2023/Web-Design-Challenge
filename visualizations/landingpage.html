SUMMARY

For this project we analyzed weather patterns versus a cities distance from the equator. We gathered the data from OpenWeatherMap API and created a dataframe with over 500 random cities.

We used Matplotlib to plot different Weather Patterns vs. latitude: Max Temperature, Cloudines, Wind Speed, and Humidity. The Data we used and the Visualizations we created can be viewed here on this website.
